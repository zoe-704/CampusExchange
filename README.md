
  # Campus Exchange

  This is a code bundle for Campus Exchange, a student marketplace website. The original design is available at https://www.figma.com/design/ocqsBZNoEbLyOYXJW0XEbN/Student-Supply-Exchange-Login.

  ## Running the code

  This website talks directly to a Supabase project (Postgres + Auth + Storage).
  See [SETUP.md](./SETUP.md) for the full setup — starting Supabase locally,
  environment variables, and applying migrations — before `npm run dev` will
  work.
